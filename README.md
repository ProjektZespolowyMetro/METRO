# METRO

### .env powinien znajdowac sie w folderze backend z formatem:
``
SECRET_KEY=key
``

### aby odpalic api nalezy wpisac
``
python backend\manage.py runserver
``

### aby odpalic frontend nalezy wpisac
``
cd frontend
npm start
``

### Frontend odpala sie na localhost:3000 podczas gdy django na 127.0.0.1:8000
